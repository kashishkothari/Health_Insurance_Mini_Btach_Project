📌 Project Overview:

This project applies Mini Batch K-Means Clustering, an efficient variant of the K-Means algorithm, to a Health Insurance dataset. The objective is to segment individuals into meaningful groups based on demographic and health-related attributes, enabling better customer profiling and data-driven decision-making.

Mini Batch K-Means is particularly suitable for large datasets as it significantly reduces computation time while maintaining clustering performance comparable to traditional K-Means.


 🎯 Objectives:

- Perform Exploratory Data Analysis (EDA) on the health insurance dataset.
- Clean and preprocess the data.
- Encode categorical variables.
- Scale numerical features.
- Apply Mini Batch K-Means clustering.
- Identify meaningful customer segments.
- Visualize and interpret clustering results.
